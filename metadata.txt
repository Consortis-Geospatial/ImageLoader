about=ImageLoader is a QGIS plugin that allows users to click on polygons (e.g., grid tiles) to automatically load and display associated raster images from a local folder. 
 
 The plugin supports customizable filename patterns using prefixes and suffixes, and provides an interactive interface for selecting the  input layer, field, and image directory.
